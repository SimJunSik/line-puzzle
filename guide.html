한 줄 퍼즐 플레이 가이드

이 가이드에서는 한 줄 퍼즐의 기본 조작법부터 고급 기능까지 게임을 즐기는 데 필요한 모든 정보를 안내합니다.

목차

  • 기본 규칙
  • 조작 방법
  • 화면 구성
  • AI 챌린지
  • 랭킹 시스템
  • 기록 향상 팁

기본 규칙

한 줄 퍼즐의 목표는 간단합니다: 시작점(START)에서 도착점(GOAL)까지 벽에 닿지 않고 선을 그리는 것입니다.

성공 조건

  • 시작점에서 출발해 도착점에 도달하면 클리어입니다.
  • 도착점에 닿는 순간 기록(클리어 시간)이 저장됩니다.
  • 제한 시간 내에 도착해야 합니다.

실패 조건

  • 벽 충돌: 그리는 선이 벽(검은색 장애물)에 닿으면 즉시 실패합니다.
  • 시간 초과: 제한 시간이 지나면 자동으로 실패 처리됩니다.
  • 경로 이탈: 시작점을 누르지 않고 그리면 시작되지 않습니다.

조작 방법

PC (마우스)

  1. 게임 시작: 녹색 START 원 위에 마우스 커서를 올리고 클릭합니다.
  2. 선 그리기: 마우스 버튼을 누른 상태로 드래그하며 경로를 그립니다.
  3. 게임 종료: GOAL 원에 도달하면 자동으로 종료됩니다.

모바일/태블릿 (터치)

  1. 게임 시작: 녹색 START 원을 손가락으로 터치합니다.
  2. 선 그리기: 손가락을 떼지 않고 화면을 드래그합니다.
  3. 게임 종료: GOAL 원에 도달하면 자동으로 종료됩니다.

버튼 기능

버튼 기능
다시 시작 현재 레벨을 처음부터 다시 시작합니다.
이전 레벨 이전 레벨로 이동합니다.
다음 레벨 다음 레벨로 이동합니다.
레벨 선택 원하는 레벨을 직접 선택할 수 있습니다.
AI 플레이 AI가 해당 레벨을 플레이하는 모습을 시청합니다.

화면 구성

게임 화면 요소

  • 제한 시간: 해당 레벨에 설정된 최대 플레이 시간입니다.
  • 남은 시간: 현재 남은 시간을 실시간으로 표시합니다.
  • 상태: 현재 게임 상태(대기/진행 중/성공/실패)를 나타냅니다.

맵 요소

  • START (녹색 원): 게임 시작 지점입니다. 여기를 눌러 시작합니다.
  • GOAL (빨간색 원): 도착 지점입니다. 여기에 도달하면 클리어입니다.
  • 벽 (검은색 영역): 피해야 할 장애물입니다. 닿으면 실패합니다.
  • 경로 (흰색 영역): 이동 가능한 구간입니다.

AI 챌린지 시스템

각 레벨에는 AI가 계산한 기준 클리어 시간이 있습니다. 이 시간보다 빠르게 클리어하면 해당 레벨에서 AI에게 승리한 것입니다.

AI 속도 설정

AI 플레이 시청 시 속도를 조절할 수 있습니다:

  • 빠름: AI가 빠른 속도로 경로를 그립니다.
  • 보통: 중간 속도로 플레이합니다.
  • 천천히: 경로를 자세히 관찰할 수 있도록 느리게 플레이합니다.

AI 기록 활용법

AI 기록은 단순한 목표가 아니라 학습 도구입니다. "AI 플레이" 버튼을 눌러 AI가 어떤 경로로 이동하는지 관찰하면 최적의 경로를 파악하는 데 도움이 됩니다.

랭킹 시스템

로컬 랭킹

개인 기록은 브라우저의 로컬 스토리지에 자동 저장됩니다. 같은 브라우저에서는 기록이 유지되며, 언제든 자신의 최고 기록을 확인할 수 있습니다.

글로벌 랭킹

글로벌 랭킹에 참여하려면 닉네임을 입력하고 "기록 제출" 버튼을 누르면 됩니다. 전 세계 플레이어들과 기록을 비교하고 경쟁할 수 있습니다.

주의: 글로벌 랭킹에 등록된 닉네임과 기록은 공개됩니다. 개인정보가 포함되지 않은 닉네임 사용을 권장합니다.

기록 향상 팁

1. 경로 미리 계획하기

시작하기 전에 START에서 GOAL까지의 경로를 눈으로 먼저 따라가 보세요. 머릿속에 경로가 그려진 상태로 시작하면 훨씬 빠르게 진행할 수 있습니다.

2. 불필요한 움직임 줄이기

복잡하게 돌아가는 경로보다 직선에 가까운 경로가 빠릅니다. 벽을 피하면서도 최대한 짧은 거리를 이동하세요.

3. 일정한 속도 유지

너무 빠르게 움직이면 벽에 충돌하기 쉽고, 너무 느리면 시간이 오래 걸립니다. 안정적이고 일정한 속도를 유지하는 것이 좋습니다.

4. AI 플레이 관찰하기

막히는 레벨이 있다면 "AI 플레이" 기능으로 최적 경로를 확인하세요. AI가 어떤 각도로 벽을 피하는지 관찰하면 도움이 됩니다.

5. 반복 연습

같은 레벨을 여러 번 반복하면 손이 경로를 기억하게 됩니다. 처음에는 클리어에 집중하고, 익숙해지면 속도를 높여 보세요.

준비되셨나요?

이제 기본 규칙과 조작법을 모두 알게 되었습니다. 게임을 시작해서 직접 플레이해 보세요!
